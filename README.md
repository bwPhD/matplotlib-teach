# Matplotlib 可视化教学系统

一个基于 Streamlit 的交互式 Matplotlib 学习平台，提供全面的图表参数教学和实时预览功能。

## 🎯 项目特色

- **🎨 交互式学习**：实时调整参数，立即预览效果
- **📚 全面覆盖**：涵盖 Matplotlib 所有主要参数
- **🌟 智能引导**：从基础到高级的教学路径
- **🚀 自动唤醒**：Streamlit Cloud 应用自动唤醒系统

## 📖 章节目录

1. **生态全景** - 了解 Python 可视化生态
2. **Matplotlib 核心解构** - 理解 Matplotlib 架构
3. **基础笔触** - 掌握基本绘图元素
4. **布局与美学** - 控制图表外观和布局
5. **进阶画廊** - 高级图表示例
6. **其他库实战** - Plotly、Altair 等库对比
7. **进阶挑战：大师之路** - 高难度实战挑战
8. **小白交互编辑练习** - 综合练习模块

## 🚀 部署说明

### 在线访问

应用已部署在 Streamlit Cloud： [访问应用](https://your-app-name.streamlit.app)

### 本地运行

```bash
# 克隆项目
git clone https://github.com/your-username/matplotlib-teach.git
cd matplotlib-teach

# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

- **前端框架**：Streamlit
- **可视化库**：Matplotlib
- **辅助库**：NumPy, Pandas

## 📊 功能特性

### 交互式参数调整
- 实时预览参数效果
- 智能参数验证

### 全面参数覆盖
- **Line2D**：线型、线宽、颜色、标记等
- **Patches**：形状、填充、边框等
- **Text**：字体、大小、样式等
- **Axes**：坐标轴、网格、边框等
- **Figure**：画布尺寸、DPI、背景等
